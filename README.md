# 🧳 Smart Tourist Itinerary Planner

A Java-based desktop application that helps users create personalized travel itineraries based on their destination, trip duration, budget, and interests. The application provides an easy-to-use graphical interface for planning trips efficiently.

---

## 📌 Project Overview

The **Smart Tourist Itinerary Planner** is designed to simplify travel planning by allowing users to generate customized itineraries through a Java Swing-based interface.

The application demonstrates object-oriented programming concepts, GUI development, and Java programming principles.

---

## ✨ Features

- 🗺️ Personalized travel itinerary generation
- 📅 Day-wise trip planning
- 💰 Budget-based recommendations
- 📍 Destination selection
- 🏨 Hotel and accommodation suggestions
- 🍽️ Restaurant recommendations
- 🎯 Tourist attraction suggestions
- 🖥️ User-friendly Java Swing GUI

## 🚀 How to Run

### Prerequisites

- Java JDK 17 or later
- VS Code or Eclipse
- Java Extension Pack (for VS Code)

### Steps

1. Clone the repository

```bash
git clone https://github.com/AISH-code-ops/SmartTourist-Itinerary-planner.git
```

2. Open the project in VS Code or Eclipse.

3. Compile the project.

4. Run the main Java file (for example, `Main.java` or `MainFrame.java`).
